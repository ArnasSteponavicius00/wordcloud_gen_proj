Wordcloud Generator
===============================================
Arnas Steponavicius - G00361891 - Data Structures and Algorithms

A java command line application that parses a file or URL based on user input
and outputs an image file of the pasrsed words.

FEATURES:
==============================================
- User inputs file or URL they wish to parse
- Specify the amount of words user wants to display in the image
- Input the name of the image they wish to input

HOW TO USE:
==============================================

- Put wordcloud.jar into any folder
- add ignorewords.txt if you wish to filter out common words *optional*
- add src folder into the same folder
- run the following command

--------------------------------------------
java -cp wordcloud.jar ie.gmit.sw.Runner
--------------------------------------------
